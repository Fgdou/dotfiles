# Dotfiles

Using GNU stow to handle all dotfiles in my linux configurations.

# Install stow
```shell
sudo pacman -Sy stow
```

# Clone
```sh
git clone https://github.com/Fgdou/dotfiles
cd dotfiles
```

# Apply
```sh
make
```
